# Firebase setup

## Firestore rules
Copiez le contenu de firebase-rules.rules dans la console Firebase > Firestore Database > Rules.

## Storage rules
Copiez le contenu de storage-rules.rules dans la console Firebase > Storage > Rules.

## Étapes complémentaires
1. Créez un document dans Firestore dans la collection admins avec l'ID correspondant à votre UID Firebase.
2. Connectez-vous avec le compte administrateur autorisé.
3. Testez l'ajout/suppression de photos depuis la page admin.
